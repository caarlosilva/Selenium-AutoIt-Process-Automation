# Selenium-AutoIt-Process-Automation

This project aim to automate some web process using Selenium and AutoIt with C#.

## Libraries used:

-[ Selenium ](https://www.seleniumhq.org/)
-[ Auto It ](https://www.autoitscript.com/site/)
-[ Google Drive API v3 ](https://developers.google.com/drive/api/v3/reference)
-[ Media Type Map ](https://github.com/samuelneff/MimeTypeMap)
-[ Json.NET ](https://www.newtonsoft.com/json)
